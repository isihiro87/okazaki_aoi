# 岡崎市葵倫理法人会 デジタル基盤（okazaki_aoi）

公開サイト（Phase 1）と公式LINE実装一式。設計は [`設計書.md`](./設計書.md) を参照。

## 構成

| パス | 内容 |
|---|---|
| `index.html` / `css/` / `js/` | 公開サイト本体。`js/data.js` の `SCHEDULE`/`REPORTS` を編集して更新 |
| `line/` | 公式LINE実装（あいさつ文・登録フォームLIFF・GAS保存先・リッチメニュー） |
| `設計書.md` | Webサイト・LINE・連携・運用の統合設計書 |

## 公開サイトの更新

`js/data.js` の配列（講話スケジュール・活動報告）を書き換えるだけで反映されます（広報委員が編集する唯一のファイル）。

## 公式LINE

セットアップ手順は [`line/セットアップ手順.md`](./line/セットアップ手順.md)、全体像は [`line/README.md`](./line/README.md) を参照。

> ⚠️ チャネルアクセストークン・チャネルシークレット等の**機密情報はこのリポジトリにコミットしない**こと（設計書 §5.2）。

## 注意

- 会員の個人情報（出席・役職・連絡先等）はこのリポジトリには含めない。役員会資料など個人情報を含むものは別管理。

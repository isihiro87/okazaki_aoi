# 公式LINE 定型文（あいさつ・案内）

LINE Official Account Manager に貼り付けて使う文面集です。
会名・URL・QR等は実値に差し替えてください。配信ルール（設計書 §3.4）：1通2スクロール以内／絵文字2個まで／敬体統一。

```
▼ 講師予定・会場・体験参加のご案内はこちら
https://okazaki-aoi.vercel.app/

■ はじめて参加してみたい方（体験参加）
このトークに「体験参加希望」と送ってください。担当より日程をご案内します。

■ 会員の方
下のメニュー「会員登録」から、お名前・役職・委員会のご登録をお願いします。
（出欠連絡や委員会のご案内に使用します）
```

## ② 会員登録のお願い（リッチメニュー誘導／個別送信用）

```
会員の皆さまへ。
公式LINEのご利用にあたり、会員情報のご登録をお願いします。

下のメニュー「会員登録」をタップ → お名前・倫理法人会での役職・所属委員会をご入力ください。
1分で完了します。役職や委員会が変わった際も、同じ画面から更新できます。
```
